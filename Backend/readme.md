# FastAPI Backend

This repository contains a minimal FastAPI backend used for learning and experimenting.

Today covered: API Endpoints and the complete setup.

## Overview

- **Project:** FastAPI Backend (learning project)
- **Language:** Python
- **Framework:** FastAPI

## Prerequisites

- Python 3.10+ installed
- git
- pip (or use a virtual environment manager)

## Setup (Windows)

1. Clone the repo and change directory:

```bash
git clone <https://github.com/Student-MuhammadAbubakar/Fast-API-Projects>
cd Backend
```

2. Create and activate a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1   # PowerShell
# or
.\.venv\Scripts\activate.bat   # cmd.exe
```

3. Install required packages:

```bash
pip install fastapi uvicorn
# Install any other project-specific dependencies (e.g. scalar_fastapi) if needed
```

4. Run the server:

```bash
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

## Setup (macOS / Linux)

```bash
git clone <your-repo-url>
cd Backend
python3 -m venv .venv
source .venv/bin/activate
pip install fastapi uvicorn
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

## API Endpoints (covered today)

The running app exposes the following endpoints (see `app/main.py`):

- GET `/shipment`
	- Description: returns a small sample shipment contact payload.
	- Example response:

```json
{
	"Contact no": "030803",
	"Name ": "MUHAMMAD ABUBAKAR",
	"Address": "Faisalabad"
}
```

- GET `/scalar`
	- Description: returns the scalar API reference generated by `scalar_fastapi.get_scalar_api_reference` (the project includes a call to this helper in `app/main.py`). This endpoint exposes additional docs returned by that helper.

## Docs

- Open the automatic OpenAPI docs at: `http://127.0.0.1:8000/docs` (Swagger UI)
- ReDoc at: `http://127.0.0.1:8000/redoc`
- The `/scalar` endpoint returns the scalar API reference used by the project.

## Quick test with curl

```bash
curl http://127.0.0.1:8000/shipment
curl http://127.0.0.1:8000/scalar
```
